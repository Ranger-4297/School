What are IDE's?

An Integrated Development Environment (IDE) is a software application that combines all of the features and tools needed by a software developer. It's graphical in nature, which means it displays information and accepts user input through windows and controls like buttons. Examples of tools found on an IDE;

  • Syntax highlighting
  • Autocomplete/Emmet
  • Building & debugging executables
